meanings of commands i ran through in powershell and docker:

- pwd = print working directory (tells me which folder im in right now).
- dir = directory listing (shows me what is inside the folder im  currently in. Eg: pennacool-intern-exercise has docker-compose, app (index.php), dockerfile, readme, db, bridge).
- cd = change directory (moves me into another folder that i tell it to...go into this folder)
- docker-compose up --build = starts up the docker app/ reads the docker-compose.yml/ starts all containers web and db, connecting them together [builds and starts up the whole docker app])
- docker-compose down = shuts down everything 
- docker ps = docker process status (shows me what containers are currently running)
- ctrl + C (in docker app) = stops running a command
